# Smart Healthcare Monitoring System

## Overview
An IoT-enabled healthcare monitoring solution built with Enterprise JavaBeans (EJB) to capture patient vitals in real time, process data securely, and generate proactive alerts for healthcare providers.

## Features
- Real-time patient data capture via IoT devices  
- Scalable EJB-based architecture for large deployments  
- Secure data storage and transmission  
- Automated alerts for anomalies in patient health  
- Analytical dashboard for healthcare providers  

## Tech Stack
- Java (EJB, J2EE)  
- MySQL Database  
- IoT device integration  
- NetBeans IDE  

## Setup
1. Clone the repository.  
2. Configure MySQL database and update credentials in `config/`.  
3. Deploy EJB components on an application server (e.g., GlassFish).  
4. Connect IoT devices to the central server.  

## Security
- SSL/TLS for data transmission  
- Role-based access control  
- Regular backups and failover mechanisms
